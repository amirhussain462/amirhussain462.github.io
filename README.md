# Amir Hussain Shaik Premium Portfolio with Admin Panel

## What is included
- Premium one-page portfolio design
- All sections scroll on the same page
- Data-driven content from `data/site.json`
- Admin panel files in `/admin`
- Decap CMS configuration for editing projects, photos, certificates, gallery and text

## Important
GitHub Pages is static hosting. The website will display normally on GitHub Pages, but the `/admin` login needs an OAuth backend.

Best easy setup:
1. Push this folder to your GitHub repository.
2. Connect the same repository to Netlify.
3. Enable Netlify Identity.
4. Enable Git Gateway.
5. Open `/admin` on the Netlify website.
6. Add projects/photos from the admin panel.

If you continue only with GitHub Pages, the public website works, but admin login will need a GitHub OAuth provider configuration.
